Projects

MAGE — Custom Vulkan Graphics Framework

A C++ graphics framework built from scratch on Vulkan, designed as a foundation for my own emulators and rendering projects.

  • Wraps device setup, swapchain recreation, command buffers, descriptor pools, and texture loading behind small RAII classes with builder-style configuration.
  • Renders textured 3D scenes via global UBOs, push constants, and a camera supporting both perspective and orthographic projection.
  • Powers my Chip-8 emulator and will be the rendering backbone for my SNES emulator (final-year project).

Chip-8 Emulator

A complete Chip-8 emulator in C++, written to validate my custom Vulkan framework against a real workload.

  • Implements all 35 opcodes with independent 960 Hz CPU and 60 Hz timer loops and full keypad input.
  • Dispatches opcodes via function-pointer tables instead of a switch.
  • Uploads the 64×32 framebuffer to a Vulkan texture each frame and samples it in a Slang fragment shader.

PongGL — OpenGL Pong (C++)

A from-scratch Pong recreation — my entry point into graphics programming, building a full game loop without an engine.

  • 2D rendering pipeline built directly on OpenGL and C++.
  • Manual AABB collision detection and physics calculations.
  • Two-player game loop and gameplay logic implemented from scratch.

Nada Aqui — Slay the Spire 2 Mod

A custom character mod for Slay the Spire 2, built on Alchyr's mod template and BaseLib, designed around three interlocking mechanics. Currently in early development.

  • Hype: a momentum counter built during combat that gates or amplifies certain cards.
  • Combo cards: "joke" cards that set up "punchline" payoffs, rewarding sequencing across turns.
  • Gambling: percentage-based card and relic effects centered on risk/reward.
